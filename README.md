# TicTakToe
Tic Tak Toe Game

# Requirements
Python==2.1.2

Make sure Python and Pip already installed in system.
# Setup

git clone https://github.com/gaurav139v/TicTakToe

Move to tictaktoe directory
pip install -r Requirement.txt

To start the Game:
python tictactoe.py
